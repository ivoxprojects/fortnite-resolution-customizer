## System requirements
- OS: Windows 7-11
- Python 3.11 or above (No need to install python when using the exe version)
- Packages needed
  - customtkinter
  - configparser
  - os
  - ctypes
  - sys
- For windows desktop only

## Features
- Cutomize your in-game resolution easily
- Clean GUI to navigate through
